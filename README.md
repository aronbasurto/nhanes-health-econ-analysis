# NHANES Health Economics Analysis

This repository contains a Stata-based analysis using publicly available
NHANES data to examine the relationship between insurance coverage and
health care utilization among U.S. adults.

## Research Question
How does insurance coverage relate to health care utilization among adults,
particularly those with chronic conditions?

## Data
Data are from the National Health and Nutrition Examination Survey (NHANES),
2017â€“2018 cycle. Raw data files are not included due to file size and CDC
distribution guidelines.

## Methods
The analysis includes:
- Merging multiple NHANES survey files
- Construction of insurance, chronic condition, and utilization measures
- Survey-weighted descriptive statistics
- Regression analysis using NHANES complex survey design
